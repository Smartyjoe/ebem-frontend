
  # Premium Minimalist Ecommerce Website

  This is a code bundle for Premium Minimalist Ecommerce Website. The original project is available at https://www.figma.com/design/q1qmC4fCsRamGE360H8w09/Premium-Minimalist-Ecommerce-Website.

  ## Running the frontend

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Running the backend API

  1. Create `backend/.env` from `backend/.env.example`.
  2. Install backend dependencies: `cd backend && npm i`.
  3. Start backend: `npm run dev:api` (from project root) or `npm run dev` inside `backend`.

  API base for frontend defaults to `http://localhost:4000/api/v1`.
  Override with `VITE_API_BASE_URL` in root `.env`.
